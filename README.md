# training_vue

## Docker Install
[Docker For Mac](https://docs.docker.com/docker-for-mac/install/#install-and-run-docker-for-mac)


## Docker build & up
```
$ ./bin/setup
```

### login to container
```
$ docker exec -it training_vue_node bash
```

### down to container
```
docker-compose down
```

### up to container
```
docker-compose up
```
or
```
docker-compose up -d
```
